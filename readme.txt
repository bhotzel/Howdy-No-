=== Howdy No! ===
Tags: Howdy No!, plugin
Tested up to: 4.8

== Description ==
A wordpress plugin for getting rid of Howdy and changing it with another word.

Replace Howdy with another word.

Install by
1. Upload the entire `howdy-no` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Now Howdy No! is prepared to use.

= Usage =

1. Go to Settings > Howdy No!.
2. Write a word that you want to replace Howdy with in the text box.
3. Hit the save button.
4. Refresh your browser.
